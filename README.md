Este programa permite la conexión de propietarios de un conjunto residencial para la realización de reuniones virtuales.
Una vez descargado ejecute "npm run dev" tanto en el directorio src como en el cliente.
En el raiz del proyecto ubique el archivo ".env" con la siguiente información:
  PORT=3000
  JWT_SECRET_KEY=
  ANNOUNCED_IP=127.0.0.1
  IP=0.0.0.0
  SMTP_USER= (correo oficial de la unidad inmobiliaria)
  SMTP_PASS= (contraseña de aplicacion)
